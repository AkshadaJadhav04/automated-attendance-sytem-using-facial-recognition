Automated Attendance System Using Face Recognition
Overview

Automated Attendance System Using Face Recognition is an AI-based project that detects and recognizes student faces through a webcam and marks attendance automatically in real time. The system reduces manual work, improves accuracy, and prevents proxy attendance.

Technologies Used

Python
OpenCV
Face Recognition
NumPy
Pandas
SQLite/MySQL
Twilio API

Features

Real-time face recognition
Automatic attendance marking
Student dataset creation
Attendance report generation
SMS alerts for absent students
Timetable-based attendance
Mask detection support

Project Workflow

Capture student face images
Train the face recognition model
Detect and recognize faces through webcam
Store attendance records automatically
Send SMS notifications for absent students
